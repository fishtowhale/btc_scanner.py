# BTC 5-Min Scanner

Simple Python scanner for BTCUSDT (5-min timeframe).

## Features
- EMA Trend Detection (9, 21, 50)
- RSI Momentum (7)
- Volume Confirmation
- Outputs: UP / DOWN / NULL

## Usage
Run:
python btc_scanner.py

## Example Output
Signal: UP  
RSI: 62  
Volume: High  
Reason: Trend + RSI + Volume
